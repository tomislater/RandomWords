Overview
--------

RandomWords is a useful package for generate random words, nicks, e-mails and lorem ipsum.

Install
-------

Quick way::

    pip install RandomWords

or::

    git clone https://github.com/tomislater/RandomWords.git
    cd RandomWords
    python setup.py install
